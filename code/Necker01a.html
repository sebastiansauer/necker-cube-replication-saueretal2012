3.3 Flag suspicious person

Lets flag persons as suspicous if ONE of the follow conditions are met

  1. RT < 300 ms in one trial (doesnt matter which)
  2. RT > 10 sec in one trial (as previous research is suspicious of time far above 3 seconds)

By this scheme, we will also capture persons with very few or with hella lot of trials.

d_long3 <- d_long2 %>% 
  ungroup() %>% 
  mutate(is_suspicious_trial = case_when(
    rt < 300 ~ 1,
    rt > 10000 ~ 1,
    TRUE ~ 0
  ))

Lets check the distribution of suspicious persons:

d_suspects <- d_long3 %>%
